* 제로보드 최근 게시물 출력 파일 설명서

- 최근 게시물 추출의 경우 스킨 디렉토리에 계속해서 하부 디렉토리로 생성할수 있습니다.
  같은 형태인데 색깔이 다를 경우 사용하면 됩니다.
  하부 디렉토리 생성시 스킨 이름을 /a/b 이런식으로 미리 경로까지 지정하시면 됩니다.

- 스킨의 약어
  최근 게시물 출력은 main.html 을 통해서 이루어집니다.
  main.html 안에는 다음의 약속어를 이용할수 있습니다.

	1. 일반 게시물 의 최근 목록
		[dir]  : 최근 게시물 출력 스킨의 image 디렉토리입니다. ex) <img src=[dir]a.gif border=0>
		[title] : 제목

		[notice_subject] : 목록중 하나를 내용까지 보여줄때 제목을 대체합니다.
		[notice_date] : 목록중 하나를 내용까지 보여줄때 날짜을 대체합니다.
		[notice_memo] : 목록중 하나를 내용까지 보여줄때 내용을 대체합니다.

		[name] : 이름을 표시합니다. (쪽지함의 링크까지 되어 있습니다)
		[memo] : 쪽지 화면 표시입니다. 메모는 절대 [loop] 사이에 넣으시면 안됩니다. 딱 한번만 출력이 됩니다.
		[subject] : 제목입니다.
		[comment] : 코멘트 수입니다. * [1] 등으로 표시되며 없으면 표시되지 않습니다.
		[loop]..[/loop] : 모든 최근 게시물 출력파일에서 반복되는 부분을 표시합니다.
	  				      [loop] 사이의 문장으로 목록을 만듭니다.
	                      반복되어서 출력될 목록을 [loop]와 [/loop] 사이에 넣으시면 됩니다.

		*[notice는 다음과 같은 형식에서 사용됩니다.
		 ------
		 제  목
		 내  용
		 ------
		 목  록
		 ------
		 목  록..

	2. 설문 조사의 최근 목록
		[dir]  : 최근 게시물 출력 스킨의 image 디렉토리입니다. ex) <img src=[dir]a.gif border=0>
		[title] : 제목
		[main_subject] : 설문 주제입니다.
		[main_vote] : 전체 설문현황입니다.  * [1] 등으로 표시되며 없으면 표시되지 않습니다.
		[subject] : 설문 항목입니다.
		[vote] : 설문 현황의 숫자입니다.  * [1] 등으로 표시되며 없으면 표시되지 않습니다.
		[loop]..[/loop] : 모든 최근 게시물 출력파일에서 반복되는 부분을 표시합니다.
	  				      [loop] 사이의 문장으로 목록을 만듭니다.
	                      반복되어서 출력될 목록을 [loop]와 [/loop] 사이에 넣으시면 됩니다.

	3. 갤러리 최근 목록
		[title] : 제목
		[dir] : 최근 게시물 출력 스킨의 image 디렉토리입니다. ex) <img src=[dir]a.gif border=0>
		[img] : 출력된 이미지를 입력합니다.
		* 갤러리 스킨에서는 [loop] 약어가 사용되지 않습니다.
		  구현의 편의성을 위해서 [img]약어에 선택된 이미지들이 표시가 됩니다.
