관리자 페이지 요약

2. 메뉴는 왼쪽의 그룹별 관리 메뉴와 상단의 전체 관리 메뉴가 있습니다

2.

2. 위 그림은 상단 전체 관리 메뉴의 그림입니다

2. 전체 관리 메뉴는 최고 관리자만이 사용할수 있습니다

2. Uninstall : 제로보드의 DB와 파일을 모두 제거하는 메뉴입니다

2. DB 백업 : 제로보드가 사용하고 있는 DB를 백업하는 기능입니다

2. DB 상태보기 : 제로보드가 사용하고 있는 DB의 상태를 볼수 있습니다

2. 첨부파일 정리 : 첨부파일을 정리하고 쓰레기 파일을 삭제할수 있습니다

2. 세션 비우기 : 인증시 필요한 세션파일을 모두 지울수 있습니다

2. 게시물 추적 : 전체 게시판에서 키워들 이용하여 게시물 추적을 할수 있습니다

2.

2. 왼쪽의 그룹 관리 메뉴입니다

2. 그룹을 생성한후 선택이 되면 하위 메뉴들이 나타납니다

2. 그룹설정: 그룹 설정을 할수 있습니다

2. 삭제: 그룹을 삭제합니다

2. 회원관리: 회원의 목록을 보고 관리할수 있고 전체메일을 보낼수 있습니다

2. 가입폼: 회원 가입폼의 항목을 설정합니다

2. 게시판 관리: 게시판의 목록을 보고 관리를 할수 있습니다

2. 추가: 게시판을 추가합니다

2. 새 그룹 추가: 새로운 그룹을 추가할수 있습니다
